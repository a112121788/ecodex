# 快捷键

ECodex 的快捷键会随功能迭代继续扩展。当前重点快捷键如下：

| 快捷键 | 行为 |
|---|---|
| `Ctrl+Shift+,` | 重载 `ecodex.json`，刷新命令面板。 |
| `Ctrl+Shift+U` | 跳转到最新未读通知，并标记为已读。 |
| `Ctrl+Shift+O` | 刷新恢复绑定并定位第一个可恢复 Pane。 |

更多命令可通过以下方式查看：

```powershell
ecodex help
ecodex completion powershell
```

设置窗口与命令面板会展示当前可用动作。若快捷键与系统或终端应用冲突，以终端输入优先。
